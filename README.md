# 코코포리아 로그 변환기 (Ccfolia Log Converter)

TRPG 세션 플랫폼 '코코포리아(Cocofolia)'의 세션 로그를 백업하고, 소장용/웹 게시용으로 변환해주는 웹 기반 유틸리티입니다.

기존의 파일 업로드 방식뿐만 아니라, **방 ID만으로 로그를 자동으로 불러오는 기능**이 추가되었습니다.

👉 **[사용하러 가기 (GitHub Pages 링크)]**

## 📢 주요 업데이트

### 1. 📡 Room ID 연동 (자동 불러오기)
코코포리아 **방 주소(URL)의 뒷부분(Room ID)**만 입력하면, 캐릭터 프로필, 대사, 이미지를 자동으로 수집합니다.

### 2. 🖼️ 이미지 영구 보존
로그에 포함된 캐릭터 프로필과 표정 이미지를 **HTML 파일 내부에 내장(Base64)**합니다.
* 코코포리아 원본 링크가 만료되어도 로그에서는 **이미지가 깨지지 않습니다.**
* 인터넷이 없는 환경에서도 이미지가 완벽하게 표시됩니다.

### 3. 📝 웹 업로드 최적화
블로그나 게시판에 로그를 올릴 때 스타일이 깨지지 않도록 **HTML 태그를 정제하고 스타일을 격리(Scoping)**하는 옵션이 추가되었습니다.

## 🚀 사용 방법

### 방법 A: Room ID로 불러오기 (추천)
1.  **[시작하기]** 탭에서 `📡 Room ID로 불러오기`를 선택합니다.
2.  코코포리아 방 URL의 끝부분(ID)을 입력합니다. (예: `ccfolia.com/rooms/THIS_PART`)
3.  **[데이터 불러오기]** 버튼을 누르면 로그 수집이 시작됩니다.

### 방법 B: 로그 파일 업로드
1.  코코포리아 방 설정에서 로그를 `.html` 형태로 다운로드합니다.
2.  **[시작하기]** 탭에서 `📁 파일 업로드`를 선택하고 파일을 엽니다.

## 🎨 출력 스타일 소개

### 📚 소설 스타일
세션 로그를 웹소설처럼 가독성 좋게 다듬는 데 최적화된 스타일입니다.
-   **특징**: 대사와 서사가 명확히 구분되며, 텍스트 가독성이 높습니다.
-   **옵션**: '가독성 향상' 체크 시 문단 들여쓰기와 자간이 조정됩니다.

### ⏱️ 타임라인 스타일
시간 순서대로 로그를 보여주는 스타일입니다.
-   **특징**: 캐릭터 프로필 이미지와 함께 대화 흐름을 파악하기 좋습니다.
-   **옵션**: 프로필 이미지를 코코포리아 원본에서 자동으로 가져오거나, 직접 업로드할 수 있습니다.

### 💬 코코포리아 디자인
원본 코코포리아 채팅창의 UI와 느낌을 최대한 재현한 스타일입니다.
-   **특징**: 실제 플레이 화면과 가장 유사한 형태로 로그를 보관할 수 있습니다.

## 🛠️ 상세 기능

| 기능 | 설명 |
| :--- | :--- |
| **🖼️ 삽화 추가** | 로그 중간 원하는 위치에 컷씬이나 이미지를 삽입할 수 있습니다. (파일/URL 지원) |
| **✂️ 파일 분할** | 로그가 너무 길 경우, 용량(KB)이나 개수 단위로 파일을 나누어 저장합니다. |
| **💾 설정 저장** | 내가 설정한 색상 테마, 폰트 등을 프리셋 파일로 저장하고 불러올 수 있습니다. |
| **🌈 테마 커스텀** | 캐릭터별 이름 색상, 배경색, 시스템 메시지 색상 등을 자유롭게 변경 가능합니다. |
